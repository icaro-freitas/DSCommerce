## Tecnologias utilizadas:
- Spring Boot
- Banco de dados H2
- Banco de dados PostgreSQL
- Jakarta Bean Validation
- JPA/Hibernate 
- Spring Security
- OAuth2
- JUnit
- TDD
- Mockito
- MockMvc
- Jacoco
- REST Assured
